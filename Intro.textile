h1. Getting set up 

h2. Learn how we set up the project and the toolchain

---

First, we need to set up your project and the toolchain. After reading this guide, you'll understand how to set up JBoss Developer Studio and JBoss Forge, and how to start a new project. Topics covered include:

* Make sure you have the right version of Java installed
* Download and install JBDS
* Download and install JBoss AS
* Download and install Maven

And of course the guide will take you through the boring business of configuration.

For those of you who prefer to watch and learn, the included video shows you how we performed all the steps.
